Most dishwashers won't run at all unless the door latch is fully engaged, so a latch that's slightly misaligned or worn can look exactly like a dead control board. We check the latch and the door switch first, along with the breaker, since all three produce the same "nothing happens" symptom and only one requires a real repair.

It's also worth double-checking that a delay start hasn't been set accidentally — a surprisingly common cause of a dishwasher that seems unresponsive but is actually just waiting. If the door and power both check out, we test the control board and touchpad directly rather than replacing the whole panel on a guess.
